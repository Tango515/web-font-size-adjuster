# 商店发布资料

## 基础信息

扩展名称：网页字体大小调整器

短描述：按网站调整并记住网页文字大小，尽量保持原网页样式。

类别建议：Accessibility / 辅助功能

成熟内容：否

远程代码：否

## 详细描述

网页字体大小调整器可以帮助你在不同网站上设置更适合阅读的文字大小。你可以在当前页面即时放大或缩小文字，也可以开启“记住此网站”，让扩展在下次打开同一主域名的网站时自动恢复之前的字号设置。

扩展默认按主域名保存设置。例如，`wx.zsxq.com` 和 `articles.zsxq.com` 会共用 `zsxq.com` 的字体大小设置。这样同一个服务下的不同子域名可以获得一致的阅读体验。

主要功能：

- 通过滑杆调整当前网页字体大小
- 支持 80% 到 200% 的字号比例
- 支持 A- / A+ 快速微调
- 支持常用比例快捷按钮
- 支持按主域名记住设置
- 支持重置当前网站设置
- 支持用户允许后的本地 HTML 文件页面

隐私说明：

扩展不上传网页内容，不读取账号、Cookie 或表单数据，不接入广告、分析或远程日志服务。扩展只保存网站标识和字体大小比例，用于下次访问时自动恢复设置。

技术说明：

扩展使用 Manifest V3，不加载远程代码。字体调整在用户浏览器本地完成，只修改页面文本元素的 `font-size`，不主动修改颜色、背景、图片、边框或其他视觉样式。由于网页布局本身依赖字体大小，放大文字后个别页面可能出现换行或布局变化。

## 权限说明

`storage`：保存用户为网站设置的字体大小比例。

`activeTab`：在用户点击扩展时获取当前标签页并发送调整指令。

`scripting`：在当前页面尚未加载内容脚本时注入脚本，以便立即调整字体大小。

`http://*/*`、`https://*/*`：在网页打开时自动应用已保存的网站字体大小设置。

`file:///*`：支持用户在扩展详情页允许后，对本地保存的 HTML 页面应用字体大小设置。

## 隐私表单建议

Single purpose：

按网站调整网页文字大小，并在用户选择记住设置后自动恢复该网站的字体大小偏好。

Remote code：

No. The extension does not load or execute remotely hosted code.

Data collection：

不收集用户个人数据。仅保存本地偏好设置，包括网站标识、字号比例和更新时间，用于恢复用户设置。

## 认证测试说明

测试账号：不需要。

测试步骤：

1. 安装扩展。
2. 打开任意网页。
3. 点击扩展图标。
4. 使用滑杆或 A- / A+ 调整文字大小。
5. 开启“记住此网站”。
6. 刷新页面，确认字号设置自动恢复。
7. 点击“重置”，确认当前网站恢复默认字号。

本扩展不需要登录，不访问远程服务，不加载远程代码。
