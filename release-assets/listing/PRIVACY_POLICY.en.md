# Privacy Policy

Effective date: 2026-05-26

Web Font Size Adjuster is a browser extension that adjusts webpage text size by website and saves the user's font size preference when the user chooses to remember a site.

## Data Stored

This extension does not collect, sell, transmit, or share personal information.

The extension only stores the following preference data locally in the browser or in browser sync storage:

- Website identifier, such as `zsxq.com`
- Font size scale selected by the user for that website
- Last updated time

This data is used only to restore the user's font size setting when they revisit the same website.

## Webpage Content

This extension does not read, upload, sell, or share webpage body content, form content, account information, browsing history, or cookies.

The extension modifies the `font-size` style of some text elements on the current page to adjust text size. This processing happens only locally in the user's browser.

## Remote Code

This extension does not load or execute any remotely hosted code.

## Third-Party Services

This extension does not use analytics, advertising, tracking, or remote logging services.

## Permission Usage

- `storage`: Saves the user's font size preference for websites.
- `activeTab`: Reads the current tab after the user clicks the extension and sends adjustment commands.
- `scripting`: Injects the content script when needed to adjust font size on the current page.
- `http://*/*`, `https://*/*`, `file:///*`: Automatically applies saved font size settings on webpages and on local files when the user allows file URL access.
