- [ ] 使用 `package/font-size-adjuster-1.0.0.zip` 上传。
- [ ] 扩展名称：网页字体大小调整器。
- [ ] 版本号：1.0.0。
- [ ] 类别：Accessibility / 辅助功能。
- [ ] 成熟内容：否。
- [ ] 远程代码：否。
- [ ] 不需要测试账号。
- [ ] 隐私政策 URL 已公开可访问。
- [ ] 英文隐私政策 URL 已公开可访问。
- [ ] 中文隐私政策 URL 已公开可访问。

- [ ] 英文商店介绍已填写。
- [ ] 中文商店介绍已填写。

## 权限说明

- [ ] `storage`：保存网站字号偏好。
- [ ] `activeTab`：用户点击扩展后读取当前标签页并发送调整指令。
- [ ] `scripting`：当前页面未加载内容脚本时注入脚本。
- [ ] `http://*/*` / `https://*/*`：网页打开时自动应用已保存设置。
- [ ] `file:///*`：用户允许后支持本地 HTML 文件。

## 隐私说明

- [ ] 不收集个人信息。
- [ ] 不上传网页内容。
- [ ] 不读取账号、Cookie 或表单内容。
- [ ] 不接入广告、分析或远程日志。
- [ ] 只保存网站标识、字号比例、更新时间。
